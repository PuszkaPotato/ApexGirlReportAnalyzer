# Contributing

Thanks for your interest in contributing to this project!  
Contributions are welcome and appreciated.

This project is primarily intended as a learning, portfolio, and experimental
project. The code is public to encourage discussion, improvement, and
collaboration.

## What you can contribute

You are welcome to:
- Fix bugs
- Improve code quality or performance
- Refactor existing code
- Improve documentation
- Suggest or implement small features

If you plan a larger change, please open an issue first to discuss it.
