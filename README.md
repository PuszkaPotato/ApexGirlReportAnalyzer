# ApexGirl Report Analyzer

Battle report analysis API for Apex Girl mobile game using OpenAI Vision API.

## Tech Stack
- .NET 10.0
- ASP.NET Core Web API
- Entity Framework Core
- PostgreSQL
- OpenAI API
